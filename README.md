# XCURENET 앱 백업 저장소 (유지)

## 구성 파일들

- `.chainlit/` - Chainlit 설정 및 UI 관련 파일들  
- `chainlit_app/` - Chainlit 애플리케이션 코드  
- `llm_conn/` - LLM 연결 관련 모듈  
- `main_agent/` - 메인 에이전트 코드  
- `middleware/` - 미들웨어 모듈  
- `sub_agents/` - 서브 에이전트 모듈  

## 변경 사항

이 저장소는 기존 `xcurenet-app-backup`에서 이름이 변경되었습니다. 모든 브랜치와 히스토리가 유지됩니다.
